# VSCDB Dark
VSCode dubeg's dark theme.


## Theme definition
```
{
    "name": "VSCDB Dark",
    "type": "dark",
    "colors": {
        ...
    },
    "tokenColors": [
        ...
    ]
}
```


## Overrides 
User settings: `AppData/.../settings.json`
```json
{
    "workbench.colorCustomizations": {
        ...
    },
    "editor.tokenColorCustomizations" : {
        "textMateRules": [
            ...
        ]
    }
}

```


## Docs
- Docs: https://code.visualstudio.com/docs/getstarted/theme-color-reference
